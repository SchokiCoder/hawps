# HAWPS

Silly program to simulate physics in *very* convincing ways.  
It'll be great. Trust me.

What does this acronym stand for now?  
Half assed wannabe physics simulator, and the name is the game.

# Install

```
git clone https://github.com/SchokiCoder/hawps.git;
cd hawps;
sudo make install;
```

You may need to replace "sudo" with "doas".  
The install can be undone by that Makefile's `make remove`.
