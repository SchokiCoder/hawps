# Contributing

If you want to report a bug, open a new [issue][issues].

If you want to see a new feature, please do the same,
before working on an implementation.

If you wish to directly develop,
follow these steps to make it convenient for everyone:  

- have a look at [docs/todo.md][docs/todo.md] to see future goals
- open an [issue][issues], where you state your intent to work on a thing
- create a [fork][forks] on GitHub
- commit your changes
- create a [pull][pulls] request on GitHub

Thanks, and have fun.
